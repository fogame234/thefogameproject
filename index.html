  • Current ongoing projects

    Threat Intelligence Bot for Discord: GitHub Repository
    A custom bot designed to automate threat intelligence gathering and monitoring on Discord servers. It collects OSINT data, analyzes suspicious activities, and provides real-time alerts to help secure communication channels against emerging threats. As of right now it pulls in RSS feeds for both private/government sectors and recent ransomware attacks.

    Current customizations include:

    • RSS Feed Monitoring: Tracks cybersecurity news from multiple sources
    • Multiple Categories: Organizes feeds into categories (private sector, government, ransomware)
    • Country Tracking: Identifies the country of origin for each feed
    • Adaptive Rate Limiting: Prevents Discord API rate limit issues
    • Paragraph Preservation: Maintains proper formatting of articles
    • Modular Architecture: Easily extendable with new modules
    • Configurable Webhooks: Send different categories to different Discord channels

    Planned features include:

    • Telegram monitoring
    • IRC monitoring
    • Website tracking and monitoring
    • Darkweb and clearnet forum scraping
    • Automated threat intelligence reports
    • CVE tracking and notifications
